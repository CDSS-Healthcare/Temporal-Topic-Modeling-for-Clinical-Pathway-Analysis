The .py file contains the code for Temporal Topic Modeling for Clinical Pathway Analysis of the Male subset of the dataset "Diabetes 130-US Hospitals for Years 1999-2008" available at: 
Clore, J., Cios, K., DeShazo, J., & Strack, B. (2014). Diabetes 130-US Hospitals for Years 1999-2008 [Dataset]. UCI Machine Learning Repository. https://doi.org/10.24432/C5230J.
